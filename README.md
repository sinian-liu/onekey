### 快捷启动命令：s

方法一：直接运行脚本（不保存文件）
```
bash <(curl -sL https://github.com/sinian-liu/onekey/raw/main/onekey.sh)
```
方法二：下载后运行（保存到本地）
```
wget -O /root/onekey.sh https://github.com/sinian-liu/onekey/raw/main/onekey.sh && chmod +x /root/onekey.sh && /root/onekey.sh
```
